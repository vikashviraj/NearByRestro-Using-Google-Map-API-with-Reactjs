### `Tech Stack`

- React.js
  - Axios - To call APIs
  - Google Place API - To show resturants near by user location
  - Google Geometry JS API - To show Map with Marker and Information

### `Follow these steps to run project`

1. Make sure Node.js installed in your system.
2. Clone this repo in your system.
3. Open `Restro-Task` folder in VSCode or Any Code Editor or open path directory in Terminal.
4. Just Open terminal and hit command `npm install` or `yarn install`.
5. Wait untill it installing, after installation simply hit `npm start` or `yarn start`
6. Happy :) Now you can see the output.
